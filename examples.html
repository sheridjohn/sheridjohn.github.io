Javascript Exercises

This page contains a few examples of taking user input and performing calculations on that input with javascript. These exercises include calculating the difference between the smallest and largest numbers in an array, calculating the total sum of the input number, returning the smallest number in an array, reversing an array, determining whether the user input is a palindrome or not.


Difference between the smallest and largest numbers in an array

Please enter numbers seperated by commas in the input field below and hit the calculate button to return the difference between the smallest and largest numbers contained within the array you entered.



Total sum of input parameter

Please enter a number in the input field below and hit the calculate button to return the total sum of the numbers that make up the parameter you entered. For example, if 3 is the input parameter, the return value should be 6, as 1+2+3 = 6.



Return the smallest number within an array

Please enter numbers seperated by commas in the input field below and hit the calculate button to return the smallest number contained within the array you entered.



Reverse an array

Please enter numbers seperated by commas in the input field below and hit the calculate button to return the reverse of the array you entered.



Palindrome check

A palindrome is a word, phrase, or sequence that reads the same backwards as forwards. For example, "madam", "nurses run", "16461". Please enter a word, or a phrase, or a number, in the input field below and hit the calculate button to determine if what you have entered is a palindrome or not.
